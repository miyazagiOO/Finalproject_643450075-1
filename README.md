## Notes App

## คำอธิบายโปรแกรม
โปรแกรม Notes App เป็นแอปพลิเคชันบันทึกข้อความที่ช่วยให้ผู้ใช้สามารถเข้าถึงและจัดการข้อความของตนได้อย่างสะดวก โดยมีความสามารถหลัก ๆ ดังนี้
เพิ่มข้อความ: ผู้ใช้สามารถเพิ่มข้อความใหม่ที่ต้องการบันทึกลงในแอปพลิเคชันได้ทันที
แสดงรายการข้อความ: แอปพลิเคชันจะแสดงรายการข้อความทั้งหมดที่ผู้ใช้เพิ่มลงในระบบ พร้อมกับคำอธิบายและปุ่มทำงานที่เกี่ยวข้อง
แก้ไขข้อความ: ผู้ใช้สามารถแก้ไขข้อความที่มีอยู่แล้วได้ตามต้องการ
ลบข้อความ: ผู้ใช้สามารถลบข้อความที่ไม่ต้องการใช้ออกจากระบบได้โดยง่าย
โปรแกรม Notes App นี้ใช้ Firebase Firestore เป็นฐานข้อมูลเพื่อการจัดเก็บข้อความทั้งหมดของผู้ใช้ และใช้งาน Compose เป็นหน้าจอการแสดงผลและการจัดการข้อมูล

## ความเป็นมา
โปรแกรม Notes App เกิดขึ้นจากความต้องการของผู้พัฒนาที่ต้องการสร้างแอปพลิเคชันที่ช่วยให้ผู้ใช้สามารถบันทึกข้อความหรือบันทึกไอเดียต่าง ๆ ได้อย่างสะดวกและง่ายดาย 
โดยมีการพัฒนาขึ้นจากแรงบันดาลใจในการสร้างเครื่องมือที่ช่วยให้ชีวิตของผู้ใช้งานเป็นมิตรและมีประสิทธิภาพมากยิ่งขึ้น

## แนวคิด
แนวคิดหลักของโปรแกรม Notes App คือการสร้างเครื่องมือที่ช่วยให้ผู้ใช้สามารถบันทึกข้อความหรือไอเดียต่าง ๆ ได้อย่างสะดวกและมีประสิทธิภาพ 
โดยมีความยืดหยุ่นในการเพิ่ม แก้ไข และลบข้อความตามต้องการของผู้ใช้ รวมถึงการใช้ Firebase Firestore เป็นฐานข้อมูลเพื่อให้ข้อมูลของผู้ใช้ถูกจัดเก็บอย่างปลอดภัยและสามารถเข้าถึงได้ทุกเวลาและทุกที่
